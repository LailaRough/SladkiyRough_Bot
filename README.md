This Telegram Bot is written in C#. It solves the problem that many people collect recipe messages in their favorites (like me) and lose them. I decided to optimize it and implement it through a telegram bot. 
In this bot, only the admin can add recipes, delete recipes, and edit.  For the administrator, you need to insert your user ID and the bot token from the Father Bot.
Other users can only view it. The bot is still under development, I will be glad to see your comments.

Этот Телеграм Бот написан на языке C#. Он решает проблему того, что многие люди коллекционирует сообщения рецептов в избранных (как и я) и теряют их. 
Я решила это оптимизировать и реализовать через телеграм бота. В этом боте только админ может добавлять рецепты, удалять рецепты и редактировать.  
Для администратора надо  вставить свой user ID и токен бота из Father Bot. Остальным пользователям доступен только просмотр. 
Бот еще в стадии разработки, буду рада увидеть ваши комментарии.
